Live Chat Revolution gives visitiors more ways to reach you.

== Description ==
Live Chat Revolution enables your visitors more ways to reach you. Gives them option to contact you via WhatsApp, Facebook Messenger, Email, Phone Call or via custom page on your website.

= Contact options =

* PhoneCall
* Facebook Messenger
* WhatsApp
* Email
* Custom URL


= Features =
* Choose what option to show on desktop and what on mobile
* Edit all text fields shown on the widget
* Edit widget position
* Change the colors of the button

= Important =
This plugin is relying on a 3rd party as a service in the plugin:
Facebook Messenger (https://m.me/)
WhatsApp (https://web.whatsapp.com/, https://api.whatsapp.com/)


== Installation ==
1. Upload the plugin files to the `/wp-content/plugins/live-chat-revolution` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the Live Chat Revolution admin enu to configure the plugin
4. Enjoy
